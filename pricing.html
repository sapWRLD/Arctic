More info about Features
More info Basic
Het basic pakket. Hierbij krijg je een functionele (statische) website met een custom domain naam. Een website die zichzelf aanpast aan de grote van het scherm (denk aan gsm, laptop, tablet,..) en u kunt kiezen uit wel 10 verschillende templates.
More info Standard
Het standaart pakket. Hierbij krijg je een functionele (statische) website met een custom domain naam. Een website die zichzelf aanpast aan de grote van het scherm (denk aan gsm, laptop, tablet,..) en u kunt kiezen uit wel 10 verschillende templates.
More info Premium
Het premium pakket. Hierbij krijg je alles wat ook in het basic en standaart pakket zit, maar je geniet ook van:
- database integration
- de keuze tussen een statische en dynamische website
- Een SSL certificaat (HTTPS)
- Mogelijkheid tot een webshop (E-Commerce)
- mogelijkheid tot betalingen doen/ontvangen via de website
More info Custom
Hierbij krijg je alles wat ook in het basic en standaart pakket zit, maar je geniet ook van:
- database integration
- de keuze tussen een statische en dynamische website
- Een SSL certificaat (HTTPS)
- Mogelijkheid tot een webshop (E-Commerce)
- mogelijkheid tot betalingen doen/ontvangen via de website
- Google analytics (hierdoor kan je zien hoeveel bezoekers je website krijgt)
- Mogelijk tot een custom design (eigen ontwerp bijvoorbeeld)
